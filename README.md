# Meet Camera Overlay

Chrome extension that adds image overlays to your Google Meet camera feed. Overlays are visible to all meeting participants.

## Install

1. Go to `chrome://extensions`
2. Enable "Developer mode"
3. Click "Load unpacked" and select this folder

## Usage

1. Click the extension icon to add/position overlays
2. Join a Google Meet - overlays appear in your video feed

**Note:** Disable Meet's background blur/effects for best results, as overlays will otherwise be treated as background.
